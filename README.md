# Mini User List App

## Description

This is a simple web application that displays a list of users fetched from a public API.
The application allows users to search by name and view user details in a modal popup.

---

## Features

* Fetch users from a public API
* Display user name, email, and company
* Search users by name
* Show loading spinner while fetching data
* Handle errors with fallback offline data
* Display user details in a modal popup

---

## Technologies Used

* HTML
* CSS
* JavaScript
* NPM (http-server)

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/SallmaAtef/UserListApp

```

2. Install dependencies:

```bash
npm install
```

---

## How to Run

Run the following command:

```bash
npm start
```

Then open your browser and go to:

```
http://localhost:8080
```

---

## Project Structure

* index.html → Main structure
* stylesheet.css → Styling
* script.js → Application logic

---

## Notes

* If the API fails, the application will display fallback offline data.
* Make sure Node.js and NPM are installed on your machine.
