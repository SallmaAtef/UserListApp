# Mini User List App

## Description

This is a simple web application that displays a list of users fetched from a public API.
It allows users to search by name and view user details in a modal popup.

---

## Features

* Fetch users from any API
* Display name, email, and company
* Search users by name
* Loading spinner while fetching data
* Error handling with fallback data
* Modal popup for user details

---

## Technologies Used

* HTML
* CSS
* JavaScript
* NPM (http-server)

---

##  Installation

1. Clone the repository or download the ZIP file
2. Open the project folder
3. Install dependencies:

```bash
npm install
```

---

## How to Run

Run the following command:

```bash
npm start
```

Then open your browser and go to:

```
http://localhost:8080
```

---

##  Project Structure

* index.html → Main structure
* stylesheet.css → Styling
* script.js → Logic and functionality

---

## Notes

* If the API fails, the app will display fallback offline data.
* Make sure Node.js and NPM are installed on your machine.
